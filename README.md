🏥 Med-Agent AI Workflow
Candidate: Vima Vincent | Deadline: 5 May 2026  

🚀 Overview
An AI-driven booking system for Medical Labs that converts unstructured WhatsApp chats into structured database entries.  

🛠️ Features
Intelligent Intake: Uses GPT-4o-mini for NLU and Intent Extraction.  

Real-time Scheduling: Captures patient data and test types.  

WhatsApp Interface: Powered by Twilio for a seamless user experience.  

⚙️ Setup
Clone the repo.

Run npm install.

Add OPENAI_API_KEY and TWILIO_SID to your .env file.

Run node server.js.
